常见容器 存储数据的“仓库”

容器主要分为三类：序列式容器、关联式容器和容器适配器。

序列式容器 (Sequence Containers)

强调元素的线性存储。

容器 特点 适用场景
std::vector 动态数组，后端插入快，支持随机访问。 首选容器。适用于频繁读取、末尾增删。
std::list 双向链表，任意位置增删快，不支持随机访问。 频繁在中间插入/删除。
std::deque 双端队列，头尾增删都很快。 需要在两端进行操作。

用法示例

  1. std::vector —— 动态数组（最常用）
    C 对比： 相当于一个会自动扩容的 int arr[]。
  1. std::string —— 字符串（救星）
    C 对比： 彻底替代 char[] 和 char*。你再也不用担心 strcpy 溢出或 strcmp 麻烦了。
  1. std::map —— 键值对/字典
    C 对比： C 语言没有对应结构。它像是一个下标可以是任何类型的数组。\
  1. std::set —— 集合
    特点： 里面的元素会自动排序，且不允许重复。
    用法： 适合用来去重。

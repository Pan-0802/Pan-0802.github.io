一、方法的使用

1、简单方法定义（无参数返回值）
通用定义格式：
修饰符 返回值类型 方法名（参数）{
方法体
Return 结果
}

分为四种方法学习：
a.无参无返回值
b.有参无返回值
c.无参有返回值
d.有参有返回值

a.无参无返回值：
Public static void 方法名(){
方法体
}
注意事项：1）void关键字表示无返回值，写了void，就不要在方法中写return 结果
2）方法之间不能相互嵌套，方法之间是平级关系
3）方法的执行顺序只和调用顺序有关

b.有参无返回值：
Public static void 方法名(数据类型 变量名){
方法体
}

c.无参有返回值：
1）格式：
Public static 返回值类型 方法名(){
方法体
Return 结果
}
2）调用：
a.打印调用：sout(方法名())
b.赋值调用：调用完之后用一个变量接收返回值的结果!!!!!!!极力推荐
数据类型 变量名 = 方法名（）

d.有参有返回值
1）格式：
Public static 返回值类型 方法名(参数){
方法体
Return 结果
}
